# C-Programming-Practice
 
Welcome to the C Programming Practice repository! This repository contains a collection of exercises and examples designed to help you improve your C programming skills. Whether you are a beginner or looking to brush up on your C knowledge, this repository has something for you.

## Introduction

C is a powerful general-purpose programming language that is widely used for system programming, embedded systems, and as a foundation for other programming languages. This repository aims to provide a variety of practice problems and examples to help you become proficient in C programming.

## Contents:

### Basic Exercises
- **Hello World Program**
 Write a program to print "Hello, World!" on the screen.
 Modify the program to take user input and greet the user.
 
- **Arithmetic Operations**
Write a program to perform basic arithmetic operations (addition, subtraction, multiplication, division) on two user-input numbers.

 - **Even or Odd**
Write a program to check whether a given number is even or odd.

 - **Factorial Calculation**
Write a program to calculate the factorial of a given number using both iterative and recursive methods.

### Control Structures
- **Prime Number Check**
 Write a program to check whether a given number is a prime number.

- **Number Guessing Game**
Write a simple number guessing game where the program generates a random number and the user has to guess it.

- **Fibonacci Sequence**
Write a program to print the Fibonacci sequence up to n terms.

- **Pattern Printing**
Write programs to print various patterns using loops (e.g., pyramid, diamond, square).

### Functions
- **Temperature Conversion**
Write functions to convert temperatures between Celsius and Fahrenheit.

- **GCD and LCM**
 Write functions to find the Greatest Common Divisor (GCD) and Least Common Multiple (LCM) of two numbers.

### Arrays and Strings
- **Array Operations**
Write a program to perform various operations on arrays (e.g., finding the maximum/minimum element, reversing the array, sorting the array).

- **Matrix Multiplication**
Write a program to multiply two matrices.

- **String Manipulation**
 Write functions to perform various string operations (e.g., reversing a string, concatenating two strings, finding the length of a string).

### Pointers
- **Pointer Arithmetic**
Write a program to demonstrate pointer arithmetic by iterating through an array using pointers.

- **Swapping Numbers**
  Write a function to swap two numbers using pointers.

### Structures
- **Student Record Management**
Create a program to manage student records using structures. Implement functions to add, display, and search records.
- **Complex Number Operations**
  Write a program to perform arithmetic operations on complex numbers using structures.

### File I/O
- **File Copy**
Write a program to copy the contents of one file to another.

- **Text File Analysis**
Write a program to read a text file and count the number of characters, words, and lines in it.

### Advanced Topics
- **Linked List Implementation**
  Implement a singly linked list with basic operations (e.g., insertion, deletion, traversal).
- **Binary Search Tree**
  Write a program to implement a binary search tree with basic operations (e.g., insertion, deletion, in-order traversal).
- **Memory Allocation**
  Write a program to dynamically allocate memory for an array using malloc and free.

### Quiz Enhancement Ideas
- **Quiz Scoring System**
Enhance your quiz program to keep track of the user's score and display the score at the end.

- **Multiple Choice Questions**
Modify your quiz program to include multiple-choice questions where users can select an answer from a list of options.

- **Random Question Selection**
Implement a feature to randomly select questions from a pool of questions for each quiz session.

### Challenge Exercises
- **Sudoku Solver**
Write a program to solve a Sudoku puzzle using backtracking.

- **Tic-Tac-Toe Game**
Create a two-player Tic-Tac-Toe game.

- **Conway's Game of Life**
  Implement Conway's Game of Life simulation.

- **Command Line Calculator**
Create a command line calculator that can evaluate simple mathematical expressions.

- **HTTP Client**
  Write a simple HTTP client to fetch and display the contents of a web page.
