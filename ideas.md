⭐ Beginner Projects
1. Command-Line Calculator

What you learn: parsing input, basic control flow, working with functions

Extensions: add parentheses support, scientific functions, or expression parsing.

2. To-Do List (Text File Based)

What you learn: file I/O (fopen, fprintf, fscanf), simple data structures

Extensions: use structs, implement sorting, save in JSON-like format.

3. Number Guessing Game

What you learn: random numbers, loops, user input validation

Extensions: add difficulty levels, high-score tracking via files.

4. Mini Shell Commands

What you learn: system(), basic parsing, string handling

Extensions: implement built-in commands like cd, ls (using opendir/readdir).

⭐⭐ Intermediate Projects
5. Text-Based Adventure / RPG

What you learn: game loops, structs, enums, modular design

Extensions: inventory system, saving/loading game state.

6. Simple Database Engine (in a file)

What you learn: records, serialization, file offsets, binary I/O

Extensions: indexing, searching, record deletion/compaction.

7. Chat Program Using Sockets

What you learn: TCP/UDP networking, concurrency (fork/threads), client/server model

Extensions: chat rooms, usernames, encryption (XOR or AES via libs).

8. Image Manipulator (BMP)

What you learn: binary file formats, bit manipulation, memory buffers

Extensions: grayscale, blur, edge-detection filters.

⭐⭐⭐ Advanced Projects
9. Custom Memory Allocator (like malloc)

What you learn: memory management, pointers, fragmentation control

Extensions: implement best-fit, first-fit, free-list merging, etc.

10. Tiny Operating-System Kernel (Bare Metal)

What you learn: low-level hardware, bootloaders, interrupt handling

Extensions: add task switching, keyboard drivers, simple filesystem.

11. Compiler for a Mini Language

What you learn: lexing, parsing, ASTs, code gen (C or bytecode)

Extensions: build a VM to run your language.

12. Ray Tracer

What you learn: math, 3D vectors, rendering pipeline

Extensions: reflections, refractions, multi-threading.
